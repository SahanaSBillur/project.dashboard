# project.dashboard
1 Project Title and Headline: Sales Performance & Profit Analysis Dashboard
 Headline: An interactive Power BI dashboard to monitor sales, profit, discounts, customers, and shipping performance across states and customers.

2 Short Description:
This dashboard provides a comprehensive overview of sales performance, profit, customer count, and discount distribution. It highlights key KPIs such as total customers, total orders, average sales, average profit, and average discount. The report analyzes performance by ship mode, state, order ID, and customer, helping to identify the most profitable regions, high-discount customers, and the shipping method generating the highest sales. 

3 Technologies used to build dashboard :
Power BI Desktop – Used to create interactive visualizations and build the complete dashboard.
Power Query – Performed data cleaning, transformation, and shaping of raw data.
DAX (Data Analysis Expressions) – Created calculated measures such as average sales, average profit, total orders, and discount metrics.
Excel  – Served as the primary data source for importing sales and customer data.
Power BI Visuals – Used charts like bar charts, pie charts, donut charts, gauge, and line charts to represent KPIs and trends.
